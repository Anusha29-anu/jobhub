# JobHub - Modern Job Portal

A modern, responsive frontend application for a job portal platform built with React and Tailwind CSS. 

## Features Implemented
- **Home Page**: Hero section, Job search UI, Category exploration, Featured Jobs.
- **Design System**: Built with modern SaaS/Enterprise aesthetics, utilizing Tailwind `@layer` variables.
- **Component Architecture**: Reusable components (`Navbar`, `Footer`, `JobCard`).
- **Data Mocking**: Centralized static JSON dummy data for reliable UI testing.
